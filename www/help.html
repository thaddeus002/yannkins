Aide

To create a project, you simply have to add a new file in the directory "projets". This file must content the following fields :

  • PROJECT_NAME : name of the project
  • SVN_DEPOT : SVN repository of the project
  • SVN_USER : SVN user - may be empty
  • SVN_PASSWD : SVN password - may be empty
  • COMPIL : compilation command
  • TESTS_UNI : units tests command
The lines beginning with '#' are commentaries.

The "analyse.sh" command launch the analysis of all projects, and the update of "index.html" and of the pages of the projects. You can put this command in a cron task to automatically execute it every day at a selected hour. But you also can launch it manually. The html reports can then be see locally or via a web server.
