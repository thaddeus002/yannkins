Aide

Pour créer un projet, ajoutez simplement un nouveau fichier dans le répertoire "projets". Ce fichier doit contenir au minimun les champs suivants :

  • PROJECT_NAME : le nom du projet
  • SVN_DEPOT : nom du dépôt SVN du projet
  • SVN_USER : utilisateur SVN - peut être laissé vide
  • SVN_PASSWD : le mot de passe SVN - peut être laissé vide
  • COMPIL : commande de compilation
  • TESTS_UNI : commande d'éxécution des tests unitaires
Les lignes commençant par '#' sont considérés comme des commentaires.

La commande "analyse.sh" lance l'analyse de tous les projets et la mise à jour de la page index.html et des pages des projets. L'idéal consiste à placer cette commande dans une tâche cron pour la lancer automatiquement à horaires fixes. Il est aussi possible de la lancer manuellement. Il suffit alors de consulter ces pages, soit localement soit les rendant accéssibles par l'intermédiaire d'un serveur HTTP.
